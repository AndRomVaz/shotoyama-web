CLUB DEPORTIVO SHOTOYAMA — GUÍA RÁPIDA

1. Descomprime el ZIP.
2. Abre NetBeans.
3. File > Open Project.
4. Selecciona la carpeta shotoyama-web, donde está pom.xml.
5. Ejecuta ShotoyamaWebApplication.java.

http://localhost:8080

NOTA DE DESPLIEGUE
La base de datos gratuita incluida para la demostración en Render caduca a los 30 días. Para una web oficial permanente, usa una base PostgreSQL de pago o externa y compra/conecta el dominio elegido.
